\# Bandit - Linux Fundamentals (Synthesis)



\## File handling

ls -> list the files in a folder

cat -> displays the content of a file

find -> search for a file in the system

cd -> enter a folder

pwd -> display where you are currently

cp -> copy a file while keeping the original

mv -> move or rename a file

rm -> delete a file





\## Creation of files/folders

mkdir -> create a folder

nano -> create a file or open it if it already exists





\## Permissions 

chmod -> modify the permissions of a file (read / write / execute)

umask -> sets the default permissions for newly created files





\## Remote access

ssh -> connect to a remote machine via secure terminal

nc -> connect to a port of a machine





\## Text processing

grep -> search for a word in a file

sort -> sort lines

strings -> extract readable text from a binary file

uniq -> delete the duplicate files

xxd -> analyze binary files





\## Compression / Decompression 

tar -> archive multiple files into a single .tar file/ extract tar archives

gzip / gunzip -> compress or decompress a single file (.gz format)

zip / unzip -> compress or extract .zip archives 

bzip2 -> compress a single file using the .bz2 format

bunzip2 -> decompress .bz2 files

file -> identify the real type of a file
